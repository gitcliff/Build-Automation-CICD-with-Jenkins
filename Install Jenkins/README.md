#  Install Jenkins


# 📖 Summary

Worked on **Install Jenkins** (branch: *feature/Install-Jenkins*), focusing on **cloud provisioning, containerized Jenkins deployment, and secure access management**.
Tasks included **creating a DigitalOcean Droplet, installing Jenkins using Docker, and retrieving the initial admin password** to solve **the challenge of setting up a scalable, persistent Jenkins server remotely**.
Set up **Jenkins inside a Docker container on an Ubuntu Droplet**, enabling automated, containerized CI/CD pipeline management accessible via a secured web interface.

---

# 🛠️ Tools Used

* **DigitalOcean Droplet**: ✨ *Provisioned and managed the remote Ubuntu server*
* **Docker**: ⚙️ *Used to deploy Jenkins container with persistent volume storage*
* **Jenkins LTS Docker Image**: 🚀 *Provided a stable Jenkins server for continuous integration workflows*

---

# 🎯 Skills Gained

* **Remote Server Management via SSH**: 💡 *Connected and administered remote Ubuntu servers securely*
* **Containerized Jenkins Deployment**: 🔍 *Installed and ran Jenkins within Docker for easy management and persistence*
* **Data Persistence and Secret Management**: 🚀 *Inspected Docker volumes and retrieved Jenkins secrets to unlock the service*

---

# ⚠️ Challenges Faced

* **Docker Not Installed Initially**: 🛠️ *Identified and installed Docker on the server before deploying Jenkins*
* **Accessing Jenkins Initial Password**: 🔄 *Learned to retrieve admin credentials from inside the container and host filesystem*
