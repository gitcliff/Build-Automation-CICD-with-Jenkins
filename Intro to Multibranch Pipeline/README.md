#  Intro to Multibranch Pipeline


# 📖 Summary

Worked on **Intro to Multibranch Pipeline**, focusing on **Jenkins Multibranch Pipeline creation, Git integration, and branch-aware Jenkinsfile configuration**.
Tasks included setting up a Jenkins Multibranch Pipeline to automatically detect and build branches from a Git repository, and configuring Jenkinsfiles with conditional logic for branch-specific CI/CD workflows.
Set up Jenkins and Github integration to automate pipeline scanning, build execution, and deployment workflows, aiming to improve pipeline automation and branch management.

---

# 🛠️ Tools Used

* **Jenkins** ✨ – *Used to create and manage Multibranch Pipeline jobs and automate CI/CD processes*
* **Git/GitHub** ⚙️ – *Source control system hosting multiple branches, integrated with Jenkins for automated scanning*
* **Groovy (Jenkinsfile)** 🚀 – *Scripted pipeline defining stages and conditional execution logic based on branch name*

---

# 🎯 Skills Gained

* **Multibranch Pipeline Setup** 💡 – *Created pipeline jobs that automatically detect and build all repository branches*
* **Branch-specific CI/CD Logic** 🔍 – *Configured Jenkinsfiles with conditional `when` blocks to execute stages based on branch context*
* **Pipeline Scanning & Indexing** 🚀 – *Monitored Jenkins branch indexing and logs to ensure proper pipeline setup and build triggers*

---

# ⚠️ Challenges Faced

* **Branch Detection Accuracy** 🛠️ – *Used regex filters (`.*`) to ensure all branches are included in the scan*
* **Conditional Pipeline Execution** 🔄 – *Implemented `when` expressions to restrict build and deploy stages to the master branch only*
